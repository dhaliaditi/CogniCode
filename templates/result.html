How to Interpret These Scores

Complexity (trustworthy)

The model achieves κ = 0.52 against expert consensus. A score above 3 means the code is structurally simple. Scores below 2.5 suggest high structural complexity that may hinder understanding.

Refactoring (useful screening)

When probability exceeds 60%, the model flags code correctly about two-thirds of the time. Use this to prioritise which files to review manually, not to make final decisions.

Readability (guide only)

The model reaches 34% of expert consensus (κ = 0.24). It detects documentation quality well but cannot judge naming clarity or idiomatic style. Treat as a rough indicator.

Maintainability (guide only)

The model reaches 42% of expert consensus (κ = 0.32). It misses architectural signals that human reviewers rely on. Use alongside manual review, not as a substitute.
